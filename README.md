# Personal Loan Propensity Streamlit App (Styled)

This app builds a full **marketing & data science dashboard** for the Universal Bank
personal loan dataset, with an enhanced, more immersive UI.

## Files

- `app.py` – main Streamlit application (entry point on Streamlit Cloud).
- `UniversalBank.csv` – source dataset used for insights and model training.
- `requirements.txt` – Python dependencies (no version pins).

## Run locally

```bash
pip install -r requirements.txt
streamlit run app.py
```
